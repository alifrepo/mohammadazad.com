Alfuix

ALFUIX.COM
Github npm JSdelivr

What it is

An open-source, CSS UI toolkit with advanced clipping & modern utilities designed for building futuristic, high-density, and tactical user interfaces.

Why I built it

Most modern design systems are built for standard, minimalist SaaS & dashboard. I wanted to break the mold and create a high-performance fui system optimized for advanced css clipping, cyber-security dashboards, and sci-fi data tracking.

How I did it

As the sole Designer and Frontend Architect, I built a cohesive ecosystem matching advanced Figma components 1:1 with lightweight, utility-first SCSS and CSS3 code for seamless developer handoff.

Alif - Islamic App

alif-islamic.app
Alif App Alif Website

What it is

A comprehensive, completely free, and 100% ad-free mobile app featuring the Quran, Hadith, precise prayer times, interactive Hajj & Umrah guides, an Islamic Quiz, a digital Tasbih counter, accurate Qibla direction, the 99 Names of Allah, and an intuitive "I'm Feeling..." Duas feature tailored to your emotional state.

Why I built it

Most Islamic apps are cluttered with intrusive ads and distracting interfaces. I wanted to design a digital sanctuary—an elegant, respectful, and distraction-free experience that prioritizes premium typography and user privacy.

How I did it

I led the UX/UI strategy, designing clean readability scales for complex Arabic text and native mobile flows. I then collaborated directly with engineering to test and deploy a polished, high-performance application package.
